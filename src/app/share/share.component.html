Vous pouvez utiliser {{properties[0].value}} sur un écran via l'usage d'un Raspberry PI ou équivalent en suivant la procédure suivante :

  • Installer le système Raspberry OS
  • Ouvrir un terminal
  • Modifier le fichier autostart par la commande

  • Ajouter les lignes suivantes :

  • Configurer le WIFI
  • Rédémarer

Pour permettre au Raspberry de se connecter sur le premier réseau disponible vous pouvez editer le fichier wpa_supplicant.conf et ajouter une section network
